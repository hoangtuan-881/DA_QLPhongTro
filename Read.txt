Công nghệ sử dụng:
 - React 19 + TypeScript + Vite
 - TailwindCSS cho styling
 - React Router cho navigation
 - FontAwesome và Remix Icons cho icons
Cấu trúc project hoàn chỉnh:
 - 25+ trang đã được tạo (dashboard, quản lý phòng, khách thuê, thanh toán…)
 - Components tái sử dụng (Toast, Dialog, Sidebar…)
 - Routing đã cấu hình đầy đủ
 - Validation form đăng nhập/đăng ký
 - Responsive design
Cách chạy project:
1️⃣ Cài đặt dependencies:
npm install
2️⃣ Chạy development server:
npm run dev
3️⃣ Truy cập:
Mở browser và vào: http://localhost:3000
4️⃣ Build production:
npm run build
